最近的项目中频繁的用到flex布局，食用体验却是好到没话说，当然开始的时候也踩过坑，好在不是很深，多在兼容性上，也是很好解决，总之flex布局可以说是布局的首选方案了。

## 概念

### 什么是Flex布局

Flex，是 Flexible Box的缩写，大概意思就是灵活的布局，很通俗，在实用中有很好的体现

``` css
.box{
  display: flex;
}
```
